\item
Formel kann nicht jede Form haben da: z.B. Konzentration f\"allt ab, Abfall abh\"angig von Konzentration etc.

Was abl\"auft bei der graphischen Darstellung kann als Formel ausgedr\"uckt werden.
\end{itemize}
}
